# 泰语水平测试

这是微信可打开的 GitHub Pages 静态网页版本。

## 发布方式

1. 在 GitHub 新建仓库，例如 `thai-level-test-019e590f`
2. 上传本文件夹里的全部文件
3. 进入仓库 `Settings`
4. 打开 `Pages`
5. Source 选择 `Deploy from a branch`
6. Branch 选择 `main`，目录选择 `/root`
7. 保存

发布成功后，GitHub 会生成类似这样的链接：

```text
https://你的用户名.github.io/thai-level-test-019e590f/
```

把这个链接发到微信，别人点击即可打开测试。
